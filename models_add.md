# `models add`

Create a model.

**Usage**:

```console
$ causadb models add [OPTIONS]
```

**Options**:

* `--model TEXT`: The name of your model, used for accessing it later.
* `--config TEXT`: Path to a model config file (e.g. /path/to/config.json). See the docs for more information.
* `--help`: Show this message and exit.
